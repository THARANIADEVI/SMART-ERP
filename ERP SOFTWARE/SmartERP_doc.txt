1. Introduction
SmartERP is a cloud-based Billing, Inventory, Accounting, GST, and Business Management Platform inspired by TallyPrime and developed using modern Python technologies.
The system is designed to help businesses manage their daily accounting operations, inventory tracking, invoicing, GST reporting, customer management, supplier management, and financial reporting through a web-based interface.
One of the key goals of SmartERP is to replicate Tally's keyboard-first workflow, allowing accountants and operators to efficiently perform all operations without requiring a mouse.
The platform supports:
? Multi-company accounting
? Inventory management
? Billing & invoicing
? GST compliance - Dont Pay Attention for Project
? Financial reporting
? Banking operations
? Keyboard-only navigation
? Cloud deployment

2. Project Objectives
The objective of SmartERP is to provide a complete business management solution that enables organizations to:
? Manage accounting records
? Track inventory
? Generate invoices
? Monitor purchases and sales
? Handle GST calculations
? Produce financial reports
? Improve business efficiency
? Operate entirely using keyboard shortcuts

15. Keyboard-Only Navigation System
The entire application should be operable without a mouse.

Global Shortcuts
F1 = Company Selection
F2 = Change Financial Year
F3 = Company Information
F4 = Calculator
ESC = Previous Screen
CTRL + H = Dashboard
CTRL + K = Global Search
CTRL + Q = Logout

Masters
ALT + L = Create Ledger
ALT + A = Alter Ledger
ALT + G = Create Group
ALT + S = Create Stock Item
ALT + U = Create Unit

Voucher Shortcuts
F5 = Payment Voucher
F6 = Receipt Voucher
F7 = Journal Voucher
F8 = Sales Voucher
F9 = Purchase Voucher
ALT + F8 = Credit Note
ALT + F9 = Debit Note

Inventory
CTRL + I = Inventory Dashboard
CTRL + N = New Item
CTRL + E = Edit Item
CTRL + T = Stock Transfer
CTRL + R = Stock Report

Billing
CTRL + B = Create Invoice
CTRL + P = Print Invoice
CTRL + SHIFT + P = PDF Export
CTRL + M = Email Invoice

Reports
ALT + B = Balance Sheet
ALT + P = Profit & Loss
ALT + T = Trial Balance
ALT + C = Cash Flow
ALT + R = Stock Summary
ALT + X = GST Reports
